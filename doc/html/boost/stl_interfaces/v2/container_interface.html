Description

A CRTP template that one may derive from to make it easier to define container types.

The template parameter D for container_interface may be an incomplete type. Before any member of the resulting specialization of container_interface other than special member functions is referenced, D shall be complete; shall model std::derived_from<container_interface<D>>, std::semiregular, and std::forward_range; and shall contain all the nested types required in Table 72: Container requirements and, for those whose iterator nested type models std::bidirectinal_iterator, those in Table 73: Reversible container requirements.

For an object d of type D, a call to std::ranges::begin(d) shall not mutate any data members of d, and d's destructor shall end the lifetimes of the objects in [std::ranges::begin(d), std::ranges::end(d)).
